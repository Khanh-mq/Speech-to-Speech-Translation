Mô tả dự án: Nghiên cứu và phát triển hệ thống dịch thuật giọng nói không qua văn bản trung gian (Textless NLP). Xây dựng và tối ưu hóa toàn diện pipeline gồm 3 module cốt lõi: (1) wav -> unit: Trích xuất đặc trưng âm học bằng HuBERT & K-means; (2) unit -> unit: Huấn luyện mô hình dịch Transformer Big (~253 triệu tham số) trên 74.000 tệp âm thanh, tự tay tối ưu VRAM GPU và tinh chỉnh siêu tham số; (3) unit -> wav: Sử dụng Vocoder để tổng hợp lại chuỗi unit thành giọng nói tiếng Việt.
